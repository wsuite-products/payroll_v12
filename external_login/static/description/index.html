Login using Microsoft Account

Steps to follow steps before installing the module:

1. Create Application on https://apps.dev.microsoft.com. It will generate the client id and the secret key.

2. Need to assign below permission:

Delegated Permissions
  • 1. Calendars.read
  • 2. Calendars.ReadWrite
  • 3. Calendars.ReadWrite.Shared
Application Permissions
  • 1. Calendars.ReadWrite
  • 2. Calendars.ReadWrite.Shared

3. Need to select platforms as WEB.

4. Need to enter Redirect URIs: BASE URL/auth_oauth/microsoft/signin

Steps to follow steps after installing the module:

1. Enter the Client id and Secret id in the Oauth provider.

2. Go to general setting and tick on Allow external users to sign up.
